Privacy Policy

Last updated: February 17, 2026

Overview

SheetGuard ("the Add-on") is a Google Sheets add-on that provides conditional cell locking and data validation. This Privacy Policy explains what data the Add-on accesses, how it is used, and how it is protected.

Data We Access

The Add-on accesses only the data necessary to provide its functionality:

  • Cell values and headers — Read to evaluate validation rules and enforce conditional requirements.
  • Sheet names and structure — Read to identify the active sheet and display its name in the sidebar.
  • Your email address — Used to identify the spreadsheet owner for rule management permissions and Pro license verification.

Data Storage

All validation rules are stored using Google's built-in DocumentProperties service, which is tied directly to the spreadsheet. Rules are not stored on any external server.

Temporary data (rule cache, violation notifications) is stored in Google's CacheService and automatically expires within minutes.

External Requests

The Add-on makes one type of external request:

  • License verification — The spreadsheet owner's email address is sent to our license verification endpoint to check Pro subscription status. No spreadsheet content, cell values, or other user data is included in this request.

Data We Do NOT Collect

We do not:

  • Store any spreadsheet content on external servers
  • Track user behavior or analytics within the spreadsheet
  • Share any user data with third parties
  • Use cookies or tracking technologies
  • Access spreadsheets other than the one currently open

Data Sharing

We do not sell, trade, or share any user data with third parties. The only data transmitted externally is the spreadsheet owner's email for license verification, as described above.

Data Retention

Validation rules persist in the spreadsheet's DocumentProperties as long as the spreadsheet exists. Users can delete all rules at any time via the SheetGuard sidebar. Uninstalling the Add-on does not automatically delete stored rules — they can be removed manually before uninstalling.

Security

All communication between the Add-on and Google's services uses HTTPS encryption. The Add-on uses the spreadsheets.currentonly OAuth scope, which restricts access to only the spreadsheet where the Add-on is actively being used.

Google API Services User Data Policy

SheetGuard's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.

Children's Privacy

The Add-on is not directed to children under 13. We do not knowingly collect personal information from children.

Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected on this page with an updated "Last updated" date. Continued use of the Add-on after changes constitutes acceptance of the updated policy.
